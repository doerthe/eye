EYE

  • EYE is a reasoning engine supporting the RGB semantic web layers.
  • It is a semibackward reasoner enhanced with Euler path detection.
  • Via N3 it is interoperable with Cwm.
  • EYE can be installed manually on Linux, Windows and MacOSX.
  • EYE is also available in a Docker container for command line use and in a Docker container for HTTP client use.
